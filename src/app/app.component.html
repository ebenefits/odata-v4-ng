Usage Examples

In "app.module.ts" import ODataModule and then inject the ODataService where you want to use this library.

All examples in this page are based on the publicly available OData V4 sample service TripPin (SERVICE_ROOT): {{serviceRoot}}.
Metadata of this service is here: {{serviceRoot}}/$metadata.

Attention to make these examples working you need to visit the TripPin service page and accept the certificate which is surprisingly not valid.
The lack of validity is due to the certificate host name not matching the actual host name of the service.
Anyway the TripPin service has been created by Microsoft so it should be safe accepting the certificate.

To run the example queries and get the response, click on the "Execute query" button. Service response could be slow depending on workload.
